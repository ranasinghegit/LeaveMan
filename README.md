LeaveMan | Full-Stack Leave Management App

LeaveMan is a comprehensive leave management system designed to streamline employee leave requests and approvals. The application provides a user-friendly interface for employees to submit leave requests and for managers to review, approve, or reject them efficiently.

This project showcases my ability to build scalable, responsive, and maintainable web applications following MVC architecture principles.

Key Features,
Employee leave requests and history.
Manager/Admin dashboard with leave approval/rejection
Role-based authentication
Responsive UI built with Bootstrap
Data persistence with Entity Framework + MSSql
Clean project structure

Tech Stack,
Frontend: React, Vite, Bootstrap 
Backend: .NET Core Web API, Entity Framework
DataBase: MSSQL
JWT authentication

Project Goals,
Simplify leave management processes for HR and employees.
Reduce manual tracking errors and enhance transparency.
Provide a scalable solution suitable for small to medium-sized organizations.
